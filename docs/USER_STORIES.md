# MVP User Stories

## Authentication

-   User can create account using email or Google.
-   User can securely login and logout.

## Task Management

-   User can create tasks.
-   User can edit or delete tasks.
-   User can mark tasks as completed.
-   User can set due dates and priorities.

## Goal Management

-   User can create long-term goals.
-   User can attach tasks to goals.
-   User can track progress of goals.

## Habit Tracking

-   User can create habits.
-   User can mark habits as completed daily.
-   User can track streaks.

## Dashboard

-   User can view today's tasks.
-   User can see goal progress.
-   User can see habit completion status.
-   User can see AI suggestions.

## AI Assistant

-   User can ask questions about productivity.
-   User can request daily planning.
-   AI provides recommendations based on user data.

## Notifications

-   User receives reminders for tasks.
-   User receives reminders for habits.
