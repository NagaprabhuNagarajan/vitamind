# UI UX Flow

User Journey

1.  Landing Screen

-   App intro
-   Sign up / Login

2.  Onboarding

-   Add goals
-   Add habits
-   Add tasks

3.  Dashboard Displays:

-   Today's tasks
-   Habit completion
-   Goal progress
-   AI insights

4.  Task Manager

-   Create task
-   Edit task
-   Complete task

5.  Habit Tracker

-   Daily habit check-in
-   Streak tracking

6.  Goal Manager

-   Long-term goal tracking
-   Progress visualization

7.  AI Assistant

-   Chat interface
-   Ask productivity questions

8.  Profile Settings

-   Notification preferences
-   Account settings
